Resilience Engine — Entregue (build limpo)
5 sistemas novos implementados
🛡️ 1 — Idempotência por Hash de Assinatura EIP-712
internal/paymaster/idempotency.go

Interface SigLockStore — drop-in Redis via SetNX sem mudar callers
InMemorySigLock com sync.Map + TTL de 2 minutos (exatamente como o doc especifica)
Troca atômica com LoadOrStore → goroutine-safe sem mutex global
GC de entradas expiradas a cada 30 s em goroutine própria
PermitSigHash(r, s) → SHA-256(r_bytes ‖ s_bytes) — v excluído intencionalmente (não é usável para evasão trivial via flip 27↔28)
Handler mapeia ErrDuplicateSig → 409 Conflict com code: "DUPLICATE_SIG"
🔄 2 — Retry com Exponential Backoff + Full Jitter
internal/paymaster/retry.go

ExecuteWithRetry(ctx, cfg, label, operation) — 4 tentativas, base 500 ms, cap 8 s
Full Jitter (random(0, cap)) — distribui thundering-herd após recovery do RPC
ErrNonRetryable — signer 4xx sobe imediatamente sem backoff (sig inválida, permit expirado)
Context-aware — respeita cancelamento durante sleep do backoff
Após esgotar tentativas → DB status='dlq', relayDLQ.push()
⚡ 3 — Relay Batcher Concorrente + Encoder Multicall3
internal/paymaster/batcher.go

POST /v1/gas/relay
   ↓ DB insert (durabilidade)
   ↓ RelayBatcher.Enqueue(id) ← non-blocking, cap 256
RelayBatcher goroutine:
   collectBatch: ≤5 items OU 500ms (whichever first)
   dispatchBatch: semaphore 3 goroutines concorrentes
   cada goroutine: ExecuteWithRetry → signer /hd/transfer
Fallback poller (5s): ListRetryableRelays → Enqueue

Multicall3 (PAYMASTER_MULTICALL_CONTRACT): quando ≥2 relays no batch, encoda aggregate3(Call3[]) inline (sem go-ethereum/accounts/abi — encoder próprio de 80 linhas). Endereço padrão 0xcA11bde05977b3631167028862bE2a173976CA11 funciona em BSC e Polygon. Economia: ~25% de gas global
🚦 4 — Rate Limit por Tier (Sliding Window por API Key)
/v1/gas/relay agora aplica antes de qualquer processamento:

Tier	Limite	HTTP
Sem chave	Bloqueado	401
sk_test_*	10/min	429 se exceder

Headers: X-RateLimit-Limit, X-RateLimit-Window, Retry-After: 60
